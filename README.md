# Advanced-Excel-Ecommerce-Dashboard
This project analyzes e-commerce sales data using an interactive Excel dashboard.

## E-commerce Sales Analysis (Advanced Excel Project)

## Overview  
This project focuses on analyzing e-commerce sales data to evaluate product performance, regional trends, discount impact, and purchasing behavior. The final Excel dashboard presents key KPIs through dynamic visuals and slicers for effective business insights.

## Tools Used  
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Charts (Bar, Column, Donut)  
- Slicers for Product and City  

## Project Workflow  
1. **Data Collection:** Imported structured e-commerce sales data into Excel.  
2. **Data Cleaning:** Removed blank rows, standardized columns, and validated numerical fields.  
3. **Pivot Analysis:** Aggregated total sales, quantity sold, and discount percentages by product and city.  
4. **Dashboard Design:** Developed an interactive dashboard with filters and KPI-driven charts.  

## Key Insights  
- **Revenue Concentration:** A limited number of products contribute a major share of total sales.  
- **Average Order Quantity:** Most transactions contain approximately **1 unit**, indicating low basket size.  
- **Discount Impact:** Discounts remain minimal (~5%) with limited influence on sales volume.  
- **Top Product Performance:** Certain products dominate revenue despite low quantity sold.  
- **City-wise Sales:** Sales performance varies significantly across cities, highlighting regional demand differences.
